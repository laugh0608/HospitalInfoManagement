## 技术架构

### 技术栈
- **后端**：Spring Boot 4.0.1 + Spring Data JPA + Gradle
- **前端**：Vue 3 + TypeScript + Vite + Element Plus
- **数据库**：SQLite (开发) / MySQL (生产)
- **安全**：Spring Security
- **其他**：Redis、WebSocket
