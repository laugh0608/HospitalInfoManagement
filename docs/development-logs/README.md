# 开发日志

本目录包含按周组织的开发日志，用于记录项目的开发进展、问题和解决方案。

## 日志命名规范

- 文件名格式：`YYYY-Www.md`
- 例如：`2026-W03.md` 表示 2026年第3周（1月13日-1月19日）的开发日志

## 周数计算

- 使用 ISO 8601 周日期标准
- 每周从周一开始，到周日结束
- 第一周是包含该年第一个周四的那一周

## 日志模板

每周日志建议包含以下内容：

```markdown
# 开发日志 - YYYY年第W周

> 时间范围：YYYY-MM-DD 至 YYYY-MM-DD

## 本周目标
- 目标1
- 目标2

## 完成情况

### 功能开发
- [x] 已完成的功能
- [ ] 未完成的功能

### 技术实现
- 具体的技术实现细节

## 遇到的问题

### 问题1：问题描述
- **问题现象**：...
- **原因分析**：...
- **解决方案**：...

## 学习与总结
- 本周学到的新知识
- 经验总结

## 下周计划
- 计划1
- 计划2
```
