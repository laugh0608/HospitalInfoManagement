## 技术选型说明

### 为什么选择 SQLite 作为开发数据库？
1. 无需安装配置独立的数据库服务
2. 数据文件本地存储，便于开发调试
3. 保留 MySQL 驱动，生产环境可轻松切换
4. 快速启动开发，降低环境配置成本

### 为什么选择 Spring Data JPA？
1. 标准的 JPA 规范，代码可移植性强
2. 强大的查询方法命名支持
3. 与 Spring Boot 深度集成
4. 良好的事务管理支持
