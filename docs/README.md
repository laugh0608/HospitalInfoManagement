# 项目文档

本目录包含 HospitalInfoManagement 项目的规划文档和开发日志。

## 目录结构

```
docs/
├── README.md                    # 本文件
├── project-plan.md              # 项目规划文档
├── database-design.md           # 数据库设计文档
├── api-design.md                # API 接口设计文档
├── security-config.md           # 安全配置说明
├── run-config.md                # 运行配置说明
├── idea-encoding-fix.md         # IDEA 编码修复
├── development-logs/            # 开发日志（按周划分）
    ├── README.md                # 开发日志说明
    └── 2026-W03.md              # 示例：2026年第3周开发日志
```

## 文档说明

### 项目规划文档
- **project-plan.md**：包含项目整体规划、功能模块、技术选型、开发计划等

### 开发日志
- 开发日志按周进行组织，文件命名格式：`YYYY-Www.md`
- 例如：`2026-W03.md` 表示 2026年第3周的开发日志
- 每周日志记录该周的开发进展、遇到的问题、解决方案等

## 如何使用

1. **项目启动阶段**：完善 `project-plan.md` 中的项目规划
2. **开发阶段**：每周创建新的开发日志文件，记录开发进展
3. **回顾阶段**：查看历史日志了解项目演进过程
