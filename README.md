# HospitalInfoManagement

社区医院病人信息管理系统

## 快速开始

### 环境要求
- JDK 25+
- Node.js 18+
- SQLite（开发环境，自动创建）/ MySQL 8.0+（生产环境）

### 后端启动

```bash
# Linux/Mac
chmod +x gradlew
./gradlew bootRun

# Windows
gradlew.bat bootRun
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
```
